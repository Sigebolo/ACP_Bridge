## 📋 任务描述

为心跳回忆 2.0 开发 Web Canvas 管理界面（基础版）。

通过 OpenClaw 的 Canvas 功能提供浏览器端管理界面，用于查看和管理 AI 伴侣。

---

## 🎨 界面需求

### 页面 1: 伴侣列表 (Home)
- 显示用户所有伴侣的卡片
- 每张卡片: 头像 + 姓名 + 原型标签 + 当前情绪状态
- 点击卡片进入伴侣详情
- 右下角 "+" 按钮 → 创建新伴侣

### 页面 2: 伴侣详情
- 基本信息 (姓名、原型、情绪)
- 操作按钮: [聊天] [日记] [相册] [设置]
- 最近记忆摘要
- 最近日记摘要 (最新3条)

### 页面 3: 日记查看器
- 按日期分页显示伴侣日记
- 每条日记: 日期 + 内容 + 爱心反应按钮

### 页面 4: 私房相册
- 网格布局显示 AI 生成的照片
- [NEW] 标签高亮未读照片
- 点击放大查看

---

## 🛠️ 技术栈

- **框架**: 纯 HTML + CSS + JavaScript (OpenClaw Canvas 兼容)
- **风格**: 现代暗色调，品牌色 #FF6B9D (粉红) + #1A1A2E (深蓝)
- **字体**: Google Fonts — Inter
- **数据源**: 读取 workspace-companions/ 下的 JSON 文件
