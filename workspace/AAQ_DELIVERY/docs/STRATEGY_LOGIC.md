# AAQ 策略逻辑说明

## 1. Pancake 策略 (双低策略)

### 核心公式
```
双低分 = 可转债价格 + (100 × 溢价率%)
```

### 筛选条件
| 条件 | 阈值 | 说明 |
|------|------|------|
| 信用评级 | ≥ AA- | 排除低评级标的 |
| 到期收益率 | > 0 | 确保正收益 |
| 双低分 | < 130 | 核心筛选指标 |
| 价格 | < 安全垫 (100) | 面值以下买入 |

### 买入逻辑
1. 扫描全市场可转债
2. 计算双低分并排序
3. 过滤掉低评级、负 YTM、高双低分标的
4. 符合条件的进入候选池 (`pancake_candidates.json`)

### 卖出逻辑
- **止盈**: 15% 收益触发卖出
- **止损**: 5% 亏损触发止损
- **时间**: 持有 30 天自动平仓

---

## 2. 安全缓冲策略 (Safety Buffer)

### 逻辑
- **买入**: 价格 < 100 (面值以下)
- **卖出**: 15% 利润 或 30 天持有期限

### 风险指标
| 指标 | 阈值 | 动作 |
|------|------|------|
| 安全缓冲触发 | 价格 < 100 | INFO 告警 |
| 均线弱势 | 价格 < MA5 且 MA5 < MA10 | WARNING 告警 |
| 均线击穿 | 价格跌破 MA10 | CRITICAL 告警 |
| 紧急止损 | 价格 < 买入价 × 0.95 | CRITICAL + 建议卖出 |

## 3. A 股尾部信号策略 (Tail Strategy)

### 信号计算
```python
price_deviation = (close - MA20) / MA20
volatility_score = volatility / rolling_volatility_50d
```

### 触发条件
- 价格偏离 > 阈值 × 日波动率 × 波动率得分
- 成交量 > 20 日均量 × 1.2

### 方向
- `price_deviation > 0` → 多头信号 (tail_signal = 1)
- `price_deviation ≤ 0` → 空头信号 (tail_signal = -1)

### 风控参数
| 参数 | 默认值 |
|------|--------|
| 仓位比例 | 10% |
| 止损 | 5% |
| 止盈 | 15% |
| 回看期 | 20 天 |
| 信号阈值 | 2.0 倍标准差 |

## 4. 协整分析 (Advanced Cointegration)

### 方法论
- **Johansen 检验**: 多资产协整关系检测
- **OU 过程**: 动态半衰期计算
- **闪电套利**: 快速回归对筛选

### 信号生成
```
hedge_ratio = -cointegration_vector[1] / cointegration_vector[0]
z_score = (spread - mean) / std
```

| Z-Score | 信号 |
|---------|------|
| < -2.0 | 强买 |
| < -1.0 | 买入 |
| > 1.0 | 卖出 |
| > 2.0 | 强卖 |

## 5. 风险评分矩阵

```
risk_score = base_risk + safety_risk + double_low_risk + ma_risk
```

| 评分区间 | 风险等级 | 仓位建议 |
|----------|----------|----------|
| 0-25 | 低风险 | 100% 仓位 |
| 26-50 | 中低风险 | 70% 仓位 |
| 51-75 | 中高风险 | 40% 仓位 |
| 76-100 | 高风险 | 拒绝开仓 |
