## 📋 任务描述

为心跳回忆平台开发完整的语音生成系统。集成阿里云百炼 Qwen TTS 作为主方案，ElevenLabs 作为备选。每个 Archetype 伴侣拥有独特的音色和语调，支持 Telegram 语音消息发送。

---

## 🎯 核心功能需求

### 1. TTS 引擎集成
- **主方案**: Qwen TTS (阿里云百炼)
  - 支持多种拟人音色
  - 支持中文和英文
  - 支持语速、音调调整
  
- **备选方案**: ElevenLabs
  - 高质量英文语音
  - 自定义音色克隆
  - 自动故障转移

### 2. Archetype 音色映射
每个 Archetype 对应独特的音色配置：

| Archetype | 音色 | 语速 | 音调 | 特征 |
|-----------|------|------|------|------|
| Guardian | 温暖男性 | 0.9 | 0.8 | 沉稳、可靠 |
| Sunshine | 活泼女性 | 1.1 | 1.2 | 开朗、热情 |
| Tsundere | 傲娇女性 | 1.0 | 1.0 | 冷漠中带温柔 |
| Dominant | 低沉男性 | 0.8 | 0.7 | 强势、自信 |
| Intellectual | 知性女性 | 0.95 | 0.9 | 理性、精准 |
| Mysterious | 神秘女性 | 0.85 | 0.95 | 神秘、优雅 |
| Comedian | 活力男性 | 1.15 | 1.1 | 幽默、轻松 |
| Romantic | 温柔女性 | 0.9 | 1.05 | 温柔、深情 |

### 3. 语音消息生成
- 文本 → 语音转换
- 自动缓存机制 (避免重复生成)
- 支持长文本分段处理
- 音频格式: OGG/MP3 (Telegram 兼容)

### 4. Telegram 集成
- 语音消息自动发送
- 支持语音消息回复
- 语音消息历史记录
- 语音消息转文本 (可选)

### 5. 性能优化
- 异步处理，不阻塞主线程
- 智能缓存策略
- CDN 分发 (可选)
- 响应时间 < 3 秒

## 📝 实现细节

### 1. TTS 引擎抽象
```javascript
class TTSEngine {
  async synthesize(text, voiceProfile) {
    // 1. 检查缓存
    // 2. 调用 TTS API
    // 3. 保存缓存
    // 4. 返回音频 URL/Buffer
  }
  
  async synthesizeWithFallback(text, voiceProfile) {
    // 尝试主方案 (Qwen)
    // 失败时自动转向备选方案 (ElevenLabs)
  }
}
```

### 2. 音色配置
```json
{
  "guardian": {
    "qwen": {
      "voice_id": "qwen_male_warm",
      "speed": 0.9,
      "pitch": 0.8
    },
    "elevenlabs": {
      "voice_id": "Adam",
      "stability": 0.5,
      "similarity_boost": 0.75
    }
  }
}
```

### 3. 缓存策略
- Key: `{text_hash}_{voice_profile_id}`
- TTL: 24 小时
- 存储: 本地文件系统 + 可选 Redis
- 清理: 定期清理过期缓存

### 4. Telegram 集成
```javascript
// 伴侣回复时自动生成语音
await telegramBot.sendVoice(chatId, audioBuffer, {
  caption: "🎤 [伴侣语音消息]",
  reply_to_message_id: messageId
});
```
